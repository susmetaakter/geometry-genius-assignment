Q-1: What is dom? What is the purpose of dom?

Answer-1:
DOM full form is Document Object Model. DOM is a web Document that is used in computer and it connects different types of programming language.

The main purpose of the DOM is that changing the document structure, style, and content. It represents the document as object or nodes that programming language can connect with the page. DOM can allow manipulation. It can control the scripting language like javascript.

Q-2:
How can you select an element from DOM?

Answer-2:
i.getElementsByTagName()
for example:const allHeadings = document.getElementsByTagName('h1');
The getElementsByTagName() returns the element that matched the passed elements.If getElementsByTagName is called upon an element, then only children of the element are searched.

ii.getElementById()
for example: const geometricDetailsContainer = document.getElementById("geometric-details-container");
The getElementById() selects an element based on its unique ID. This is the easiest way to find an HTML element in the DOM tree.

iii.getElementsByClassName()
for example: const geometryDetails = document.getElementsByClassName("geometry-details");
The getElementsByClassName() method select all the elements specific class name.

iv.querySelector()
for example: const geometricContainer = document.querySelector('.geometric-container li');
It returns the first element that matched in the document.

v.querySelectorAll()
for example: const geometricContainer = document.querySelectorAll('.geometric-container li');
It returns all the element of the document.

Q-4: What is the callback Function? What is work of callback Function?

Answer-4: A callback function is a function that is to be executed after another function execution.

A callback is a function passed into another function as an argument to be executed later.A high order function is a function that accepts another function as an argument.Callback function can be synchronous or asynchronous.
